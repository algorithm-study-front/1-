## 진행 방식
주 2회 진행

화요일
- 온라인으로 진행합니다. 
- 3문제를 미리 풀어오고 스터디를 진행하기 전까지 해당하는 문제 + 풀이를 깃헙에 올립니다.
- 스터디 시간에는 
  - 이전 Feedback 공유
  - 서로의 코드를 보고 고칠점을 찾는 시간을 갖습니다.여러가지 질문이 오가거나 지식이 오가면 좋습니다.

토요일 
- 오프라인으로 진행합니다. 
- 모여서 1시간 30분동안 3문제를 풀고 해당하는 문제 + 풀이를 깃헙에 올립니다. 
- 스터디 시간에는 
  - 이전 Feedback 공유
  - 서로의 코드를 보고 고칠점을 찾는 시간을 갖습니다. 여러가지 질문이 오가거나 지식이 오가면 좋습니다.
